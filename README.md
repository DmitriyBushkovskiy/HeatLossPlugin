# HeatLossPlugin

Программный модуль по расчету теплопотерь здания на основе данных извлеченных из ТИМ-модели для "nanoCAD BIM Строительство"

## Структура репозитория

/docs - документация (в т.ч. инструкция по работе с программным модулем)

/examples - тестовые модели, отчетная документация

/libs - библиотеки nanocad

/src - исходный код модуля
